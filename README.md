# Validação de Regras de Negócio em Java

Projeto didático para demonstrar validação de regras de negócio com exceção personalizada.

## Conceitos

- Validação no construtor
- Exceção customizada (`DomainException`)
- Tratamento com `try/catch`
- Diferença entre erro de domínio e erro de entrada

## Regras

- Idade ≥ 18
- Nome não pode ser nulo ou vazio

Projeto focado em reforçar conceitos de orientação a objetos e tratamento de exceções.
